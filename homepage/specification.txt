Homepage assignment

10+ html tags used: <section>, <article>, <figure>, <figcaption>, <p>, <div>, <nav>, <ul>, <li>, <footer>, <header>, <a>, <aside>, <video>, <h1>, <h2>, <h3>

5 CSS+ properties used: background: / border: / border-radius: / font-size: / font-family: / margin: / padding: / color: / width: / height:

Bootstrap features used: List Group, Buttons, Carousel
I utilized bootstrap to create a photo carousel, list group to create a list of links on 2 separate pages, and used their buttons to toggle a dialog.

JavaScript used
I used JavaScript addEventListeners to listen for DOMContentLoaded and click, which allowed me create a dialog toggle functions to open and close photo dialog
and also add functionality to a button in the dialog to close itself.
